


## 安装

```bash
# 拉取代码

# 进入目录


# 安装依赖
npm install

# 启动项目
npm run dev
```

会自动打开项目 http://localhost:9528

## 编译打包

```bash
# 测试环境
npm run build:stage

# 生产环境
npm run build:prod
```
